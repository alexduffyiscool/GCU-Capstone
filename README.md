This capstone project presents the development of a fitness tracking app designed to optimize workouts and nutrition for 
various fitness goals, including hypertrophy, strength training, powerlifting, and cardio. The app uses user data to track 
progress and provide real-time recommendations for progressive overload by suggesting appropriate weight increases for
exercises. Additionally, the app incorporates a macro-calorie tracking feature that calculates personalized daily 
nutritional targets, and tracks intake to ensure users meet their daily goals. This app combines automated workout 
adjustments with nutritional guidance to make a solution for people looking to improve their fitness performance with 
different goals like bodybuilding and keep track of their dietary needs easily. This repository contains the partial
frontend of the app. The entire fullstack of the project is hosted locally.
